# Zero-Shot-Deepfake-Detection
Multi-Scale Deepfake Detection Framework | Python, PyTorch, MoCo v2, Computer Vision * Developed a PyTorch-based deepfake detector using a multi-scale pyramid approach (1x, 0.5x, 0.25x) and a MoCo v2 pre-trained ResNet-50 backbone to identify structural anomalies across various AI image generators.
